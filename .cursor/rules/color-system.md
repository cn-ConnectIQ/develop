# 颜色语义系统（不能硬编码颜色值）

始终使用 Tailwind 自定义变量：
主办方功能 → text-brand-blue / bg-brand-blue-light
展商功能 → text-brand-green / bg-brand-green-light
AI 功能 → text-brand-purple / bg-brand-purple-light
待处理 → text-brand-amber / bg-brand-amber-light
VIP/积分 → text-brand-gold
错误/删除 → text-red-600 / bg-red-50

侧边栏背景：bg-sidebar（#1A1A2E）
内容区背景：bg-content（#F5F5F3）
卡片边框：border-border-light（#D3D1C7）
